# 🚀 Gerador de Posts para Redes Sociais com IA

Aplicação que usa **Claude AI (Anthropic)** para gerar posts criativos para Instagram, Twitter/X e LinkedIn.

## 🛠️ Tecnologias
- [Anthropic Claude API](https://console.anthropic.com) — modelo de IA generativa
- [Streamlit](https://streamlit.io/) — interface web interativa

## ▶️ Como Usar
1. Acesse o app no Streamlit Cloud
2. Insira sua **Anthropic API Key** na barra lateral
3. Escolha a rede social, tom e tema
4. Clique em **Gerar Post**!

## 🔑 Obter API Key
1. Acesse [console.anthropic.com](https://console.anthropic.com)
2. Crie uma conta gratuita
3. Vá em **API Keys** e gere sua chave
